# 1753 C

- https://codeforces.com/problemset/problem/1735/C
- 考虑先填小的
- 我们看这个字母前边能接什么：
	- 后边没接东西的字母
	- 不是我自己
	- 假如我后边已经有了一串，如果这一串没满26个，这我不能接上这最后一个，如果满了就可以接上这个成环，那么我前边能接的就是：后边没接东西，不是我自己，不是上边说的那个
- 贪心即可
# 1710 A

- https://codeforces.com/problemset/problem/1710/A
- 必须按行或者按列涂，我们对按列涂讨论，两者一样
- 每一种颜色必须有两列以上
- 我们优先放少的，放至少两列进去，但是不能让图只剩下一列
- 如果会让图只剩下一列，我们这个颜色少放一列，如果少放一列会导致只有一列进去，我们就不放这个颜色
