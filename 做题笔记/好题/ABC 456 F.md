# 题解： - 线段树优化 Min-Plus 矩阵 DP

## 题意转化

题目要求我们在 $N$ 天中选择若干天作为假期，且相邻两个假期之间的工作日不能超过规定天数。通过分析可知，问题的核心可以转化为：在长度为 $N$ 的序列中，对于每一个长度为 $K$ 的滑动窗口 $[l, l+K-1]$，求满足间隔条件的最小花费，并在此基础上取所有窗口的最优解。

由于每次只需考察长度为 $K$ 的区间，我们可以考虑使用动态规划（DP）计算特定区间的最小代价，再利用数据结构加速这一过程。

---

## 动态规划 (DP) 设计

对于某一个长度为 $K$ 的窗口中的第 $i$ 个元素，我们可以定义两个状态：

- $dp_0[i]$：表示处理到第 $i$ 天，且**第 $i$ 天不放假**的最小花费。
    
- $dp_1[i]$：表示处理到第 $i$ 天，且**第 $i$ 天放假**的最小花费。
    

根据题意中的间隔限制，状态转移方程为：

$$dp_0[i] = dp_1[i-1]$$

$$dp_1[i] = \min(dp_0[i-1], dp_1[i-1]) + A_i$$

直接计算的话，对于每个窗口都需要 $O(K)$ 的时间，总时间复杂度为 $O(N \cdot K)$，在 $N, K$ 较大时会超时。我们需要对其进行优化。

---

## Min-Plus 代数与矩阵优化

观察上述 DP 转移方程，其中只包含了**取最小值**和**加法**两种运算。这完美契合 **Min-Plus 代数（广义矩阵乘法）** 的性质。

在 Min-Plus 代数中：

- 传统的乘法被替换为加法 $(+)$
    
- 传统的加法被替换为取最小值 $(\min)$
    
- 单位元：加法的单位元是 $\infty$，乘法的单位元是 $0$。
    

我们可以将状态转移方程改写为矩阵形式：

$$\begin{pmatrix} dp_0[i] \\ dp_1[i] \end{pmatrix} = \begin{pmatrix} \infty & 0 \\ A_i & A_i \end{pmatrix} \otimes \begin{pmatrix} dp_0[i-1] \\ dp_1[i-1] \end{pmatrix}$$

> **验证**：
> 
> 矩阵第一行：$\min(\infty + dp_0[i-1], 0 + dp_1[i-1]) = dp_1[i-1]$
> 
> 矩阵第二行：$\min(A_i + dp_0[i-1], A_i + dp_1[i-1]) = A_i + \min(dp_0[i-1], dp_1[i-1])$
> 
> 与原转移方程完全一致！

因此，对于任意一个窗口 $[l, l+K-1]$，最终的状态向量可以通过将区间内所有的状态矩阵相乘，再乘以初始向量得出：

$$V_{end} = M_{l+K-1} \otimes M_{l+K-2} \otimes \dots \otimes M_{l} \otimes V_{start}$$

对于初始状态，初始向量为 $V_0 = \begin{pmatrix} 0 \\ A_{l-1} \end{pmatrix}$（如果 $l=1$，则 $A_0 = \infty$）。

---

## 线段树维护区间信息

既然问题转化为了**求区间矩阵连续乘积**，这正是线段树的强项。

1. **建树**：线段树的叶子节点维护单个元素的转移矩阵 $M_i = \begin{pmatrix} \infty & 0 \\ A_i & A_i \end{pmatrix}$。
    
2. **向上合并**：父节点的矩阵等于右子节点的矩阵乘以左子节点的矩阵（注意广义矩阵乘法不满足交换律，顺序必须严格为 `d[x] = d[rs] * d[ls]`）。
    
3. **查询**：对于每个窗口左端点 $l$，右端点 $r = l + K - 1$，利用线段树查询区间 $[l, r]$ 的矩阵乘积结果 `res`。
    

最终，窗口 $[l, r]$ 内的答案就是矩阵 `res` 与初始向量相乘后，取对应合法状态的最小值：

$$\text{Cost} = \min(res_{2,1} + 0, res_{2,2} + A_{l-1})$$
