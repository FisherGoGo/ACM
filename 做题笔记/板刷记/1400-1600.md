# 1418 C

- https://codeforces.com/problemset/problem/1418/C
- 设 $dp [i][0/1]$ 表示到第i个怪兽且第i个怪兽是 0我打 1朋友打 所需要的最小的点数
- $dp[i][1] =min(dp[i-][0]+k1,dp[i-2][0] + k2)$ k1 k2 是 1 的个数
- $dp[i][0] =min(dp [i-1][1],dp [i-2][1])$
- 因为第一把是朋友开始，所以 $dp[0][1]=dp[1][0]=\infty$ 因为我不能从这转移

# 2167 G

- https://codeforces.com/contest/2167/problem/G
- 考虑 DP ，我们设 $dp[i]$ 表示在不改变 $a[i]$ 的条件下前边都符合单调不降
- 那么简单转移即可
$$
dp[i]=\min\limits_{0\le j<i\,,\,a[j]\le a[i]}\{dp[j]+cost(j+1,i-1)\}
$$
- 我们令 $a[n+1]=INF$ ，那么 $dp[n+1]$ 就是答案
