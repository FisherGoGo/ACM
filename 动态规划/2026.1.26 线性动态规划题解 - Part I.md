# C

- **状态设计**：$f[i][j]$ 表示只考虑放前 $n$ 种花，一共放了 $j$ 盆的方案数
- **状态转移**：显然我们可以若我们新加一种花进来，且这种花放了 $k$ 盆，那么 $f[i][j]$ 的方案数会加 $f[i-1][j-k]$ ，因为此时等于是第 $i$ 种花放 $k$ 盆的方案数等于前 $i-1$ 种花放 $j-k$ 盆的方案数，累加即可
$$
f[i][j]=\sum\limits_{0\le k \le a_{i}} f[i-1][j-k] 
$$
- **边界条件**：$f[i][0]=1$ 一朵花都不放的方案数为 $1$

# E

- **状态设计**：$f[i][j]$ 表示把 $i$ 分成 $j$ 份的方案数
- **状态转移**：
	- 我们转移时分两种情况：有 $1$ 存在 / 最小那份大于 $1$ 
	- 首先前一个等于 $dp[i-1][j-1]$ 表示新的那份就是 $1$ 
	- 后一个等于 $dp[i-j][j]$ 表示先给每一份分一个 $1$ 上去，然后把剩下的 $i-j$ 分成 $j$ 份，保证每一份都大于 $1$ ，但是有这一个贡献的前提是 $i>j$ 
- **边界条件**：$dp[i][1]=1$ 把任意数分成一份的方案数为 $1$ ，若 $i<j$ 那么 $dp[i][j]=0$ 
> [!NOTE] 转移
> - 为什么我们可以这样子转移，主要是为了去重，我们把方案分为有 $1$ 和没 $1$ 的，通过这个划分使得两个集合去重
# F

- **状态设计**： $dp[i][j]$ 表示经过 $j$ 次传球后，球在第 $i$ 号同学手中的方案总数
- **状态转移**：显然，由于每个同学只能把球传给左边或右边的同学，那么第 $j$ 次传球后球在 $i$ 号手中，只可能来源于第 $j-1$ 次传球时，球在 $i$ 的**左邻居**或**右邻居**手中。我们将左右邻居的情况累加即可：
$$dp[i][j] = dp[left\_neighbor][j-1] + dp[right\_neighbor][j-1]$$
- **边界条件：** $dp[1][0] = 1$ ，第 $0$ 次传球时，球一定在小蛮（1号同学）手里，方案数为 $1$。其余 $dp[i][0]$ 均为 $0$ 

# G

- **状态设计：** $dp[i]$ 表示在第 $i$ 秒时，守望者所能走过的**最远距离**
- **状态转移：** 
	- 我们第一步先考虑只用闪烁，若 $m\ge 10$ ，那么 $dp[i]=dp[i-1]+60$ ，否则 $dp[i]=dp[i-1]$ ，即原地等待回蓝
	- 第二步我们看若此次跑步的收益大于休息，那么直接转移，即 $dp[i]=\max\{dp[i],dp[i-1]+17\}$ 
- **边界条件：** 初始都为 $0$ 即可
# H

- **状态设计：** $dp[i][j]$ 表示将字符串 $A$ 的前 $i$ 个字符转换为字符串 $B$ 的前 $j$ 个字符所需要的最少操作次数
- **状态转移：** 为了求得 $dp[i][j]$，我们需要考虑三种操作中代价最小的一种：
	- **删除操作：** 相当于把 $A$ 的第 $i$ 个字符删掉，即从 $dp[i-1][j]$ 转移而来，步数 $+1$
	- **插入操作：** 相当于在 $B$ 的第 $j$ 个位置插入一个字符，即从 $dp[i][j-1]$ 转移而来，步数 $+1$
	- **替换操作：** 比较 $A[i]$ 和 $B[j]$：
		- 若 $A[i] == B[j]$，不需要操作，代价为 $0$
		- 若 $A[i] \neq B[j]$，需要修改字符，代价为 $1$
		- 从 $dp[i-1][j-1]$ 转移而来，步数 $+\,tot$
- **边界条件：** 
	- 若 $i = 0$：字符串 $A$ 为空，需插入 $j$ 个字符才能变成 $B$，故返回 $j$
	- 若 $j = 0$：字符串 $B$ 为空，需删除 $A$ 中的 $i$ 个字符，故返回 $i$

# J

- **状态设计：** $dp[i]$ 表示从第 $i$ 分钟开始到工作日结束（第 $n$ 分钟）所能获得的最大空暇时间
- **状态转移：** 我们从最后时刻向前推导，对于每一分钟 $i$，有两种情况：
	1. **当前分钟没有任务开始 (`cnt[i] == 0`)：**
	    - 这意味着尼克在这一分钟是空闲的
	    - **转移方程：** $dp[i] = dp[i+1] + 1$
	2. **当前分钟有任务开始 (`cnt[i] > 0`)：**
	    - 尼克必须从开始的任务中**任选其一**执行。执行任务期间（长度为 `len`）他不能获得空暇
	    - 为了让总空暇时间最大，我们要选一个任务，使得任务结束后的那一刻起能获得的空暇时间最大
	    - **转移方程：** $dp[i] = \max(dp[i + a[num].len])$，其中 $num$ 遍历所有在第 $i$ 分钟开始的任务
- **边界条件：**$dp[n+1] = 0$ ，即工作结束后的时刻，没有空暇时间可言
