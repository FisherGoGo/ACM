# F

- 设状态 $dp[i]$ 表示体积 $i$ 能否被若干物品凑出，0 为不能，1 为能
- 初始化为 $dp[0]=1$ 
- 转移方程为 $dp[j]=dp[j]\lor dp[j-v[i]]$ ，意思是如果 $j-v[i]$ 能被凑出，那么加上当前物品 $j-v[i]+v[i]=j$ 也能够被凑出
- 最后看能被凑出的最大值即可

# H

- 设 $dp[i][j]$ 表示考虑到第 $i$ 个砝码，重量 $j$ 能不能称出来
- 如果能称出 $j$ ，那么自然能称出 $j+w[i]$ 和 $|j-w[i]|$ 
- 所以转移方程为：
	- $dp[i][j]=(dp[i-1][j]\lor dp[i-1][|j-w[i]|]\lor dp[i-1][j+w[i]])$

# K

- 基础：完全背包
- 题目是要求我们保证原来能被凑出的数在删除一些面额钱币之后还能被凑出
- 我们先从小到大排序，这样子我们在处理大面额的时候，若其能被小面额处理出来，那么会已经被标记，这样子就可以删去

```cpp
for(int i=1; i<=n; i++){
    if(f[a[i]]){       // 如果当前面额 a[i] 已经被之前的货币组合出来了
        ans--;         // 说明它是多余的，计数减 1
        continue;      // 跳过，不需要用它去更新别人
    }
    // 如果 a[i] 组合不出来，它就是必须存在的“核心货币”
    f[a[i]] = 1;       // 标记它自己可以被表示
    for(int j=a[i]; j<=a[n]; j++){ 
        f[j] = f[j] | f[j-a[i]]; // 完全背包转移：金额 j 能否由包含 a[i] 的组合表示
    }
}
```
