# 树的直径

- 树上任意两节点之间最长的简单路径即为树的「直径」
- 我们可以用两次**DFS**或者用**树形DP**来在 $O(n)$ 内求出

## 两次DFS

- 我们先从任意节点 $y$ 开始，将其能到达的最远点记为 $z$ 
- 然后从 $z$ 开始，将其能达到的最远点记为 $z'$ ，那么 $(z,z')$ 的距离为树的直径

>定理：在一棵树上，从任意节点 ![](data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7 "y") 开始进行一次 DFS，到达的距离其最远的节点 $z$ 必为直径的一端。

### 代码

```cpp
int d[N],c;
vector <int> E[N];
void dfs(int u,int fa){
	for(int i=0;i<T[u].size();i++){
		int v=T[u][i];
		if(v==fa) continue;
		d[v]=d[u]+1;
		if(d[v]>d[c]) c=v;
		dfs(v,u);
	}
}

int main(){
	dfs(1,0);
	d[c]=0;
	dfs(c,0);
	cout<<d[c];
	return 0;
}
```

## 树形DP

- 我们记录 $1$ 为根的树时，每个节点作为子树的根向下，所能延伸的最长路径长度 $d_1$ 与次长路径长度 $d_2$ 的和为直径
- 树形 DP 可以在存在负权边的情况下求解出树的直径

### 代码

```cpp
int d1[N],d2[N];
int d=0;

void dfs(int u,int fa){
	d1[u]=0;
	d2[u]=0;
	for(int i=0;i<=E[u].size();i++){
		int v==E[u][i];
		if(v==fa) continue;
		dfs(v,u);
		int t=d1[v]+1;
		if(t>d1[u]){
			d2[u]=d1[u];
			d1[u]=t;
		}
		else if(t>d2[u]){
			d2[u]=t;
		}
	}
	d=max(d,d1[u]+d2[u]);
}
```
