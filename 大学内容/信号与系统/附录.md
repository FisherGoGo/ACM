# 特征根求齐次解

- 单实根 $\lambda_{1}$：$C_{i}e^{\lambda_{i} t}$
- $r$ 重实根 $\lambda_{i}$ ：$(C_{0}+C_{1}t+\ldots+C_{r-1}t^{r-1})e^{\lambda_{i} t}$
- 共轭复根 $\alpha\pm j\beta$ ：$e^{\alpha t}[C_{1}\cos\beta t+C_{2}\sin\beta t]$
- $r$ 重复根 $\alpha\pm j\beta$ ：$e^{\alpha t}\left[\sum\limits_{i=0}^{r-1} C_{1i}t^{i}\cos\beta t+C_{2i}t^{i}\sin\beta t\right]$
# 激励求特解

- 常数 $E$ ：
	- 与特征根无重叠：$P$
	- 有重叠（k 重）：$t^{k}P$
- 多项式 $E_{i}t^{i}$ ：
	- 无重叠：$P_{i}t^{i}$
	- 有重叠：$t^{k}E_{i}t^{i}$
- 指数 $Ee^{\alpha t}$ ：
	- 无重叠： $Pe^{\alpha t}$
	- 有重叠： $Pt^{k}e^{\alpha t}$
- 正余弦：
	- 无重叠：$A\cos\omega t+B\sin\omega t$
	- 有重叠：$t^{k}[A\cos\omega t+B\sin\omega t]$ 
# 差分方程齐次解

- 单实根： $C_{i}\lambda_{i}^{k}$
- $r$ 重实根： $(C_{0}+C_{1}k+\ldots+C_{r-1}k^{r-1})\lambda_{i}^{k}$
- 共轭复根 $re^{\pm j\theta}$ ： $r^{k}[C_{1}\cos\theta k+C_{2}\sin\theta k]$
# 差分方程特解

- 常数 $E$ ：
	- 与特征根无重叠：$P$
	- 有重叠（r 重）：$k^{r}P$
- 多项式 $E_{i}t^{i}$ ：
	- 无重叠：$P_{i}k^{i}$
	- 有重叠：$k^{r}E_{i}k^{i}$
- 指数 $Ea^{k}$ ：
	- 无重叠： $Pa^{k}$
	- 有重叠： $Pk^{r}a^{k}$
- 正余弦：
	- 无重叠：$A\cos\theta k+B\sin\theta k$
	- 有重叠：$k^{r}[A\cos\theta k+B\sin\theta k]$ 

# 积化和差

- $\sin\alpha\cos\beta=\frac{1}{2}(\sin(\alpha+\beta)+\sin(\alpha-\beta))$
- $\cos\alpha\cos\beta=\frac{1}{2}(\cos(\alpha+\beta)+\cos(\alpha-\beta))$
- $\cos\alpha\sin\beta=\frac{1}{2}(\sin(\alpha+\beta)+\cos(\alpha-\beta))$
- $\sin\alpha\sin\beta=\frac{1}{2}(\cos(\alpha-\beta)-\cos(\alpha+\beta))$ 
# 和差化积

$$\sin A + \sin B = 2 \sin \frac{A+B}{2} \cos \frac{A-B}{2}$$

$$\sin A - \sin B = 2 \cos \frac{A+B}{2} \sin \frac{A-B}{2}$$

$$\cos A + \cos B = 2 \cos \frac{A+B}{2} \cos \frac{A-B}{2}$$

$$\cos A - \cos B = -2 \sin \frac{A+B}{2} \sin \frac{A-B}{2} \quad $$
# 欧拉公式

$$
\sin z= \frac{1}{2j}(e^{iz}-e^{-iz})
$$
$$
\cos z=\frac{1}{2}(e^{iz}+e^{-iz})
